# Flux OS V1.0.1

What is it?, Its a web-based based Operating System simulation

# Features

- Custom bootscreen
- A CLI Interface
- Real-time Desktop clock
- Working Nasa Uplink
- Customisable Wallpaper
- Window Transparency Settings 
- Web-App Dock

# How to use it

- Download the index.html and open it in a browser or just click this link https://the-degenerate-otaku.github.io/Flux-OS/

# Tech Stack

- HTML
- JS
